# Reference Styles

Documentation of design systems used for inspiration in the NateCue AI Builder Logs project.

---

## 1. Coding for Marketers
*Source: codingformarketers.com*

### Typography
- **Primary Font:** `JetBrains Mono` (Google Fonts)
- **Weight:** Bold (700) or Extra Bold (800) for headings; Medium (500) for body.
- **Styling:** Uppercase headings, tight line-height (~0.9-1.1).

### Color Palette
- **Background:** White (`#FFFFFF`) or Deep Zinc (`#09090b`) for dark mode.
- **Accent:** Matrix Green (`#00FF41`).
- **Borders:** Sharp 1px/2px solid black/zinc.

### Components
- **Hard Shadows:** `4px 4px 0px #000000`.
- **Grids:** 20px - 40px linear gradients.
- **Buttons:** Boxy, high contrast, non-rounded.

## 2. T2000.AI (Futuristic Retro / Cyber-Premium)
*Source: t2000.ai*

### Typography (High-End Contrast)
- **Heading Font:** `Instrument Serif`. Frequently uses *Italics* for a luxury feel within technical contexts.
- **Technical Font:** `IBM Plex Mono`. Used for UI labels, commands, and data.
- **Labels:** Uppercase, wide letter-spacing (`tracking-widest`).

### Color Palette
- **Primary Background:** Obsidian Obsidian (`#040406`).
- **Glow Accent:** Mint Green (`#00D68F`). High-glow effects on hover.
- **Muted Blue-Gray:** `#5A6070` for borders and secondary text.

### Visual Effects
- **Grain Overlay:** SVG Fractal Noise (~4% opacity) for a tactile, textured feel.
- **Atmospheric Depth:** Large, soft cyan/blue vignettes in corners.
- **Scanlines:** Subtle repeating gradients (horizontal or vertical focus).

### Components
- **Windowed UI:** Mac-style terminal windows with "traffic light" controls.
- **Thin Borders:** `1px solid rgba(255,255,255,0.1)`.
- **Geometric Icons:** Heavy use of Unicode symbols (◆, ◎, ⟳) over standard SVG icons.
